# 每日推荐

根据你的偏好每天推荐一部电影或电视剧，并通过 MoviePilot 已配置好的微信互动通知发送菜单。

## 菜单

```text
发送 /每日要：订阅
发送 /每日换：换一部
发送 /每日跳：今日跳过

不要直接回复 `1`、`2`、`3`，微信里的数字会先被 MoviePilot 自带媒体交互处理。
```

## 推荐逻辑

- 使用 TMDb Discover API 按类型、语言、题材、评分、投票数筛选候选。
- 排除已推荐过的内容。
- 可排除 MoviePilot 已订阅内容。
- 可排除 MoviePilot 已入库内容。
- 不自动下载，只在你回复 `1` 后创建 MoviePilot 订阅。

## 必填

- TMDb Read Access Token。需要填写 TMDb 后台的 `API Read Access Token`，不是普通 API Key；可填写完整 `Bearer eyJ...`，也可只填写 `eyJ...`。
- 填写 Token 后保存配置，再打开 `立即推荐一次`。执行结果会显示在插件数据页。

## 排查

- 打开 `立即推荐一次` 后如果没有立刻出推荐，先看插件数据页的“最近执行”。
- 插件会在日志里输出 TMDb 请求页数、候选数量、跳过原因、MoviePilot 识别和入库检查结果。
- 如果开启了“排除已订阅/排除已入库”，首次筛选可能需要逐个识别候选，耗时会比单纯调用 TMDb 更长。
- 微信没有收到消息时，确认插件里的“通知类型”先使用默认 `订阅`，并确认 MoviePilot 通知设置里微信渠道启用了对应类型。

## 建议配置

```text
推荐内容：电影和电视剧
语言偏好：不限，或按你的习惯选择国语/外语
类型偏好：科幻、动作、悬疑等
最低评分：6.5
最低投票数：200
推荐小时：09:00
```
